# タイムライン - 2025年〜現在

## 2025

🧠 2025年1月24日、AnthropicのComputer Useに関する重要なAPIアップデート（ヘッダー: `anthropic-beta: computer-use-2025-01-24`）がリリースされ、複雑なUI環境のナビゲーションにおけるモデルの正確性と信頼性が向上しました。

🧠 2025年1月30日、GoogleはGemini 2.0 Flashを新たなデフォルトモデルとしてリリースしました（Gemini 1.5 Flashも引き続き利用可能）。これに続き、2025年2月5日にはGemini 2.0 Proがリリースされました。

🧠 2025年2月6日、GitHubは開発者向けの新たな自律的ワークフローと生産性向上を強調する重要な発表の一環として、Copilot Agent Modeを正式に導入しました。その後、2025年2月24日には、VS Code Stable/Insidersでの利用可能化を確認する続報ブログ記事を公開し、早期アクセスを超えた初の広範なロールアウトとなりました。

🧠 2025年4月4日、GitHubはすべてのユーザーに対するAgent Modeのロールアウトを正式に発表しました。これにはModel Context Protocol (MCP) のサポートと、Copilotコードレビューエージェントの導入が含まれていました。

☁️ 2025年5月6日、9年間にわたる反復開発と無数のコミュニティ貢献を経て、Grafana k6 v1.0.0がリリースされました。

🧠 AnthropicのAI搭載コマンドラインコーディングアシスタントであるClaude Codeは、2025年2月24日にClaude 3.7 Sonnetと同時にベータリサーチプレビューとして初めて導入されました。その後、2025年5月22日に一般提供（GA）が開始されました。

🧠 2025年5月22日のClaude 4（OpusおよびSonnet）のリリースにより、ピクセルカウントの改善とレイテンシの削減を特徴とし、Computer Use機能がモデルのコアなエージェント型ワークフローにさらに統合されました。

🧠 2025年6月17日、GoogleはGemini 2.5 ProおよびFlashの一般提供を開始しました。同日、速度とコスト効率に最適化されたモデルであるGemini 2.5 Flash-Liteもあわせて発表されました。

🧠 2025年6月25日、GoogleはGemini 2.5 Pro向けのオープンソースコマンドラインインターフェースとしてGemini CLIを正式にローンチし、ターミナル環境から直接、AIを活用したコーディング、デバッグ、リサーチなどを可能にしました。

🧠 エージェントの機能を支えるGitHub Copilot Chat拡張機能が、2025年6月30日にMITライセンスの下でオープンソースソフトウェアとしてリリースされました。

🧠 2025年7月9日、Visual Studio Codeの6月安定版リリース（v1.102）において、GitHub Copilotコーディングエージェントの一般提供およびMCPサポートが実現しました。

🧠 2025年8月6日、Gemini 2.5を搭載したJulesが正式にベータ版を終了し、一般向けにローンチされました。

🧠 GPT-5はOpenAIが開発したマルチモーダル大規模言語モデルであり、同社のジェネレーティブ・プリプレイド・トランスフォーマー（GPT）基盤モデルシリーズの5番目のモデルです。シリーズのGPT-4に続くモデルとして2025年8月7日にローンチされ、推論能力と非推論機能を共通のインターフェースの下で統合しました。

🧠 2025年9月12日にリリースされたVisual Studio Codeの8月アップデート（v1.104）では、モデルの「自動（Auto）」選択が追加され、GitHub Copilot Agentにカスタム指示を提供するための `AGENTS.md` ファイルのサポートが導入されました。

🐛 機密データを収集し、アクセス可能なパッケージの悪意あるバージョンを自動的に公開する自己増殖型ワームである「Shai-Hulud」npmサプライチェーン攻撃が、2025年9月15日に初めて検出されました。

🧠 2025年9月18日にNotion 3.0がローンチされ、AI機能が「Notion AI Agents」としてリブランディングされるとともに、ワークスペース内で自律的なマルチステップのエージェント型ワークフローが導入されました。

📊 2025年9月25日にPostgreSQL 18がリリースされ、並行I/Oタスクのパフォーマンスを大幅に向上させる非同期I/O（AIO）が導入されました。

🧠 2025年9月29日、AnthropicはClaude Code v2.0によって同ツールを強化し、チェックポイント、ネイティブのVS Code拡張機能、ターミナルUXの改善、およびClaude Agent SDKを導入しました。

🧠 AnthropicのSkills API（`anthropic-beta: skills-2025-10-02`）が2025年10月2日に正式導入され、`SKILL.md` ファイルを通じて、ワークフローの指示、実行可能コード、およびドキュメントをClaude用の再利用可能な機能としてパッケージ化する標準化された手法が提供されました。

🧠 2025年10月15日、AnthropicはClaudeの機能を強化するため、「Anthropic認定（Anthropic-verified）」およびコミュニティ提供のスキルを発見・共有できる中央集権型プラットフォーム「Skills Registry」をローンチしました。

🐛 2025年10月29日、世界規模のDNS設定ミスが発生し、Microsoft 365、Xbox Live、および主要なAzure法人顧客のサービスに影響が及びました。

🧠 2025年10月22日、LangChainとLangGraphは正式にバージョン1.0に到達しました。このリリースにより、LangGraphはAPIの安定性を保証する「永続的エージェントフレームワーク（durable agent framework）」へと移行し、永続的な状態管理、ヒューマン・イン・ザ・ループ（HITL）ワークフローの組み込みサポート、および再設計されたドキュメントサイトが提供されました。

🖥️ 2025年10月29日にCursor 2.0がリリースされ、コードベース全体でのマルチエージェントワークフローの編成や高速なコード生成に最適化された、AIネイティブのコーディングインターフェース「Composer」が導入されました。

☁️ 2025年11月14日、AWS LambdaはRustを使用したサーバーレスアプリケーション構築のサポートを追加しました。

🧠 2025年11月18日、GoogleはGemini 3を正式にローンチしました。このメジャーリリースにより、強化された推論、マルチモーダル理解、およびエージェント機能が、Geminiアプリ、AI Search、Google AI Studio、Vertex AIなどに提供されました。

🧠 Gemini 3のリリースにあわせ、2025年11月18日にGoogle Antigravityプラットフォームが発表されました。これはVisual Studio Codeを大幅にカスタマイズしたAIエージェントファーストのIDEであり、GoogleのAI駆動型開発ツールへの本格参入を示すものです。

🧠 2025年11月20日、AnthropicのSkillsフレームワークにおいてModel Context Protocol (MCP) のサポートが実現し、スキルが標準化されたMCPサーバーを介して外部ツールやデータソースとシームレスに相互作用できるようになりました。

🧠 Peter Steinbergerによって作成されたオープンソースのAIエージェントフレームワーク「Clawdbot」（後にOpenClawへリブランディング）が、2025年11月24日に初めてリリースされました。リリース直後からバイラルな注目を集め、最初の24時間でGitHubスター数9,000に達しました。

🌐 2025年12月2日、Anthropicはエージェント型ワークフローや開発者ツールにランタイムを活用することを目的に、Bunの買収を発表しました。

🏢 急速に進化するAIエージェントのエコシステムに対し、中立的なガバナンスを提供するためのLinux Foundation傘下の新たな財団として、2025年12月9日にAgentic AI Foundation（AAIF）が設立されました。

🧠 GPT-5の段階的なアップデートであるGPT-5.2が2025年12月11日にリリースされ、推論能力の向上、レイテンシの削減、およびツール連携の強化が図られました。このリリースでは、後方互換性を維持しつつ、異なるユースケースに対応するより専門的なモードが導入されました。

📜 2025年のクリスマスにRuby 4.0.0がリリースされました。

## 2026

🖥️ 2026年1月7日のGemini CLI v0.23.0リリースにおいて、「Agent Skills」がプレビュー機能として初めて導入され、ユーザーが `SKILL.md` ファイルを介して専門的な知識でエージェントを拡張できるようになりました。

🧠 AIエージェントの構築、反復開発、デプロイを行うためのオールインワンのオープンソースTypeScriptフレームワークであるMastra 1.0が、2026年1月20日にリリースされました。Gatsbyの背後にあるチームによって開発され、ワークフロー、RAG、メモリ、オブザーバビリティのための統合ツールを提供します。

🖥️ Gemini CLIのSkills機能は、2026年1月27日のv0.26.0リリースにおいて正式に安定（stable）チャネルへと昇格し、あわせて新しいエージェント機能を生成するための組み込みの `skill-creator` ツールも導入されました。

🧠 Anthropicからの商標クレームを受けてClawdbotは2026年1月27日にMoltbotへ改名されました。その3日後の2026年1月30日、プロジェクトは正式にOpenClawと再度改名され、同じ時期にGitHubスター数100,000を突破しました。

🖥️ 2026年1月下旬、Gemini CLIは、Skillsフレームワークによって実現した新たなエージェント型ワークフローを補完するため、`/rewind` コマンドと永続的なツール実行ポリシーを追加しました。

🧠 Visual Studio Codeの2026年1月リリース（v1.109）にて、GitHub Copilot Agent内でのAnthropicのClaudeモデルのサポートと、強化されたマルチエージェントセッション管理が導入されました。

🧠 Visual Studio Codeの2026年2月リリース（v1.110）では、「エージェントプラグイン」、実験的な「エージェント型ブラウザツール」、およびGitHub Copilotにおいてエージェントが異なるセッション間でメモリを共有する機能が導入されました。

🧠 2026年2月23日、AnthropicのAgent SDKをベースに構築され、メッセージングアプリ統合機能を備えた、OpenClawの軽量なコンテナ化代替案であるNanoClaw v1.1.0が初のドキュメント化されたリリースとして公開され、アップストリーム更新スキルの導入と重大なパストラバーサル脆弱性の修正が行われました。

🧠 2026年3月9日、Microsoftは「Microsoft 365 Copilot Wave 3」を発表し、自律型のエージェントワークフローシステムである「Copilot Cowork」と、企業全体のエージェント管理・ガバナンス専用のコントロールプレーンである「Agent 365」を導入しました。

🧠 2026年3月12日にOpenClaw v2026.3.12がリリースされ、新しいプロバイダー・プラグインシステムと、エージェント活動を管理するための中央ダッシュボードによる、大幅なアーキテクチャの刷新が導入されました。

🧠 自律的なエージェント型ワークフロー構築用に設計された最新のAIオーケストレーションフレームワーク「Microsoft Agent Framework 1.0」が、2026年4月3日にオープンソースプロジェクトとして初めてリリースされました。

⚙️ 2026年4月12日、最新のメジャー安定版としてLinux 7.0がリリースされ、高度なハードウェアサポートやメモリ安全なプログラミング言語のさらなる統合とともに、カーネルの進化が継続されました。

🧠 2026年4月22日にNanoClaw v2.0.0がリリースされました。これは新しいエンティティモデル、デュアルデータベースによるセッション設計、およびBunベースのランナーを特徴とする、ゼロからのアーキテクチャ書き換え（リライト）でした。

🧠 OpenAIのGPT-5シリーズにおける主要なアップデートであるGPT-5.5が2026年4月23日にリリースされ、強化された推論、マルチモーダル理解、およびエージェント機能により、能力が大幅に飛躍しました。2026年4月24日にはAPIの提供開始が発表され、OpenAIのAPIエンドポイントを通じて開発者が最新モデルを利用できるようになりました。

🧠 Googleは2026年5月19日のGoogle I/OにおいてAntigravity 2.0を発表し、Gemini CLIを置き換えるGoベースのターミナルエージェントとしてAntigravity CLIを導入しました。これにより、デスクトップアプリで使用されているものと同じエージェントランタイムに対する軽量なインターフェースが提供されるようになりました。
