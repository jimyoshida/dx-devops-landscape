# タイムライン - 2025年〜現在

## 2025年

🧠 2025年1月24日、Anthropic の Computer Use に対する重要な API アップデートがリリースされ（ヘッダー: `anthropic-beta: computer-use-2025-01-24`）、複雑な UI 環境を操作する際のモデルの精度と信頼性が向上した。

🧠 2025年1月30日、Google は Gemini 2.0 Flash を新しいデフォルトモデルとしてリリースし、Gemini 1.5 Flash も引き続き利用可能とした。続いて2025年2月5日に Gemini 2.0 Pro がリリースされた。

🧠 2025年2月6日、GitHub は新しい自律型ワークフローと開発者の生産性向上を強調した大規模な発表の一環として、Copilot Agent Mode を正式に発表した。続いて2025年2月24日、GitHub は VS Code Stable/Insiders での提供を確認するフォローアップのブログ記事を公開し、アーリーアクセスを超えた最初の本格的な展開となった。

🧠 2025年4月4日、GitHub は Agent Mode の全ユーザーへの展開を正式に発表し、これには Model Context Protocol (MCP) のサポートと Copilot コードレビューエージェントの導入が含まれていた。

☁️ 2025年5月6日、9年間にわたる改良と数えきれないコミュニティの貢献を経て、Grafana k6 v1.0.0 がリリースされた。

🧠 Anthropic の AI 搭載コマンドラインコーディングアシスタントである Claude Code は、2025年2月24日に Claude 3.7 Sonnet とともにベータ版のリサーチプレビューとして初めて公開された。その後、2025年5月22日に一般提供が開始された。

🧠 2025年5月22日の Claude 4（Opus および Sonnet）のリリースにより、computer use 機能がモデルの中核的なエージェント型ワークフローにさらに統合され、ピクセルカウントの改善とレイテンシの低減が実現された。

🧠 2025年6月17日、Google は 2.5 Pro および Flash の一般提供を発表した。同日、速度とコスト効率に最適化されたモデルである Gemini 2.5 Flash-Lite も発表した。

🧠 2025年6月25日、Google は Gemini 2.5 Pro 向けのオープンソースのコマンドラインインターフェースとして Gemini CLI を正式にリリースし、ターミナル環境で AI 搭載のコーディング、デバッグ、リサーチなどを直接行えるようにした。

🧠 エージェントの機能を支える GitHub Copilot Chat 拡張機能は、2025年6月30日に MIT ライセンスの下でオープンソースソフトウェアとして公開された。

🧠 GitHub Copilot コーディングエージェントと MCP サポートの一般提供は、2025年7月9日の Visual Studio Code 6月安定版リリース（v1.102）で実現された。

🧠 2025年8月6日、Jules は正式にベータ版を終了し、Gemini 2.5 を搭載して一般公開された。

🧠 GPT-5 は OpenAI が開発したマルチモーダル大規模言語モデルであり、生成事前学習済みトランスフォーマー（GPT）基盤モデルシリーズの5番目にあたる。シリーズでは GPT-4 の後継として、2025年8月7日にリリースされ、推論機能と非推論機能を共通のインターフェースの下で統合した。

🧠 2025年9月12日にリリースされた Visual Studio Code の8月アップデート（v1.104）は、「Auto」モデル選択を追加し、GitHub Copilot Agent にカスタム指示を与えるための `AGENTS.md` ファイルのサポートを導入した。

🐛 機密データを収集し、アクセス可能なパッケージの悪意あるバージョンを自動的に公開する自己増殖型ワームである Shai-Hulud npm サプライチェーン攻撃が、2025年9月15日に初めて検出された。

🧠 2025年9月18日に Notion 3.0 がリリースされ、AI 機能を「Notion AI Agents」として再ブランド化し、ワークスペース内で自律的かつ複数ステップのエージェント型ワークフローを導入した。

📊 2025年9月25日に PostgreSQL 18 がリリースされ、非同期 I/O（AIO）を導入して並行 I/O タスクのパフォーマンスを大幅に向上させた。

🧠 2025年9月29日、Anthropic は Claude Code v2.0 でツールを強化し、チェックポイント、ネイティブの VS Code 拡張機能、ターミナル UX の改善、そして Claude Agent SDK を導入した。

🧠 2025年10月2日、Anthropic の Skills API（`anthropic-beta: skills-2025-10-02`）が正式に導入され、`SKILL.md` ファイルを介してワークフローの指示、実行可能なコード、ドキュメントを Claude 向けの再利用可能な機能としてパッケージ化する標準的な方法を提供した。

🧠 2025年10月15日、Anthropic は Skills Registry をリリースした。これは Claude の機能を拡張するための「Anthropic 検証済み」およびコミュニティ提供のスキルを発見・共有するための一元化されたプラットフォームである。

🐛 2025年10月29日のグローバルな DNS 設定ミスにより、Microsoft 365、Xbox Live、および主要な Azure 法人顧客に障害が発生した。

🧠 LangChain と LangGraph は2025年10月22日に正式にバージョン 1.0 のマイルストーンに到達した。このリリースは、API の安定性へのコミットメントを伴う「durable agent framework（永続的なエージェントフレームワーク）」への LangGraph の移行を示すものであり、永続的な状態、ヒューマン・イン・ザ・ループ（HITL）ワークフローの組み込みサポート、刷新されたドキュメントサイトを特徴としている。

🖥️ 2025年10月29日に Cursor 2.0 がリリースされ、「Composer」が導入された。これはマルチエージェントワークフローのオーケストレーションとコードベース全体にわたる高速なコード生成に最適化された AI ネイティブなコーディングインターフェースである。

☁️ 2025年11月14日、AWS Lambda は Rust を使用したサーバーレスアプリケーションの構築のサポートを追加した。

🧠 2025年11月18日、Google は Gemini 3 を正式にリリースした。これは、強化された推論、マルチモーダル理解、エージェント機能を Gemini アプリ、AI Search、Google AI Studio、Vertex AI などにもたらした大規模なリリースである。

🧠 2025年11月18日、Gemini 3 のリリースと同時に Google Antigravity プラットフォームが発表された。これは Visual Studio Code を大幅に改変した AI エージェントファーストの IDE であり、AI 搭載開発ツールへの Google の参入を示すものである。

🧠 2025年11月20日、Anthropic Skills フレームワーク内での Model Context Protocol (MCP) のサポートが実現し、スキルが標準化された MCP サーバーを介して外部ツールやデータソースとシームレスに連携できるようになった。

🧠 Peter Steinberger によって作成されたオープンソースの AI エージェントフレームワークである Clawdbot（後に OpenClaw に改名）は、2025年11月24日に初めてリリースされた。これは即座に爆発的な注目を集め、最初の24時間以内に GitHub スター9,000個に到達した。

🌐 2025年12月2日、Anthropic は Bun の買収を発表し、このランタイムを自社のエージェント型ワークフローと開発者ツールの基盤として活用することを目指した。

🏢 2025年12月9日、急速に進化する AI エージェントのエコシステムに中立的なガバナンスを提供するため、Linux Foundation 傘下の新しい財団として Agentic AI Foundation (AAIF) が設立された。

🧠 GPT-5 の漸進的なアップデートである GPT-5.2 が2025年12月11日にリリースされ、推論能力の向上、レイテンシの低減、ツール統合の強化を特徴とした。このリリースは後方互換性を維持しつつ、さまざまなユースケースに向けたより専門的なモードを導入した。

📜 Ruby 4.0.0 は2025年のクリスマスにリリースされた。

## 2026年

🖥️ 2026年1月7日、Gemini CLI の v0.23.0 リリースで「Agent Skills」がプレビュー機能として初めて導入され、ユーザーが `SKILL.md` ファイルを介してエージェントを専門的な知識で拡張できるようになった。

🧠 AI エージェントの構築、反復、デプロイのためのオールインワンのオープンソース TypeScript フレームワークである Mastra 1.0 が、2026年1月20日にリリースされた。Gatsby を手がけたチームによって開発され、ワークフロー、RAG、メモリ、可観測性のための統合ツールを提供する。

🖥️ 2026年1月27日の v0.26.0 リリースで、Gemini CLI の Skills 機能が正式に安定版チャンネルに昇格し、新しいエージェント機能を生成するための組み込み `skill-creator` ツールも導入された。

🧠 Anthropic からの商標に関する苦情を受けて、Clawdbot は2026年1月27日に Moltbot へと改名された。その3日後の2026年1月30日、プロジェクトは正式に OpenClaw へと改名され、ほぼ同時期に GitHub スター100,000個に到達した。

🖥️ 2026年1月下旬、Gemini CLI は Skills フレームワークによって可能になった新しいエージェント型ワークフローを補完するため、`/rewind` コマンドと永続的なツール実行ポリシーを追加した。

🧠 GitHub Copilot Agent 内での Anthropic の Claude モデルのサポートと、強化されたマルチエージェントセッション管理が、Visual Studio Code の2026年1月リリース（v1.109）で導入された。

🧠 Visual Studio Code の2026年2月リリース（v1.110）は、「Agent Plugins」、実験的な「Agentic Browser Tools」、および GitHub Copilot のエージェントが異なるセッション間でメモリを共有する機能を導入した。

🧠 Anthropic の Agent SDK 上に構築され、メッセージングアプリ統合を備えた OpenClaw の軽量なコンテナ化された代替手段である NanoClaw v1.1.0 が、2026年2月23日に最初の文書化されたリリースとして公開され、アップストリーム更新スキルを導入し、重大なパストラバーサル脆弱性を修正した。

🧠 2026年3月9日、Microsoft は「Microsoft 365 Copilot Wave 3」を発表し、自律型エージェントワークフローシステムである「Copilot Cowork」を、企業全体のエージェント管理とガバナンスのための専用コントロールプレーンである「Agent 365」とともに導入した。

🧠 2026年3月12日に OpenClaw v2026.3.12 がリリースされ、新しいプロバイダープラグインシステムとエージェントの活動を管理する一元化されたダッシュボードによる大きなアーキテクチャの転換が導入された。

🧠 自律型エージェントワークフローの構築のために設計された最新の AI オーケストレーションフレームワークである Microsoft Agent Framework 1.0 が、2026年4月3日にオープンソースプロジェクトとして初めてリリースされた。

⚙️ 2026年4月12日、最新のメジャー安定版として Linux 7.0 がリリースされ、高度なハードウェアサポートとメモリ安全なプログラミング言語のさらなる統合により、カーネルの進化を継続した。

🧠 2026年4月22日に NanoClaw v2.0.0 がリリースされた。これは、新しいエンティティモデル、デュアルデータベースのセッション設計、Bun ベースのランナーを特徴とする、ゼロからのアーキテクチャの書き直しである。

🧠 OpenAI の GPT-5 シリーズの大規模なアップデートである GPT-5.5 が2026年4月23日にリリースされ、推論、マルチモーダル理解、エージェント機能の向上により能力の大きな飛躍を示した。API の提供は2026年4月24日に発表され、OpenAI の API エンドポイントを通じて最新モデルを開発者に提供した。

🧠 2026年5月19日、Google は Google I/O で Antigravity 2.0 を発表し、Gemini CLI を置き換える Go ベースのターミナルエージェントとして Antigravity CLI を導入した。これはデスクトップアプリで使用されているのと同じエージェントランタイムへの軽量なインターフェースを提供する。
