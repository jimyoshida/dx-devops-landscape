For each section file's level 2 heading, list the relevant DSS-P skills in the "Relevant DSS-P Skills" block section.
The skills are listed in the format of "Skill Number. Skill Name".
The skill number is the same as the one in the DSS-P skill tree, and the skill name is the same as the one in the DSS-P skill tree.

<Example>

File: website/docs/skills/section04.md

```markdown
## Governance, Risk, and Compliance (GRC)

> **Relevant DSS-P Skills:**
>
> - 4\. Security > 4\.1 Security Management > Security Management
> - 4\. Security > 4\.1 Security Management > Privacy Protection
```

</Example>
