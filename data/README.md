# Metadata

This directory contains the section index `sections.yml` and digest file `digest.md`.

The section index is used to auto-update the section titles across the following files:

- `website/docs/skills/section*.md` — skill files (primary targets)
- `website/docs/class-policy.md` — section title references in the skill mapping table
- `website/docs/intro.md` — section title references in the domain list

The digest file is used to create the mind map and to inform agents the structure.
